# Contributing

This is a Snowflake Sales Engineering demo asset. It is not a general-purpose
open-source project accepting external contributions.

## Internal contributions

Contributions go through internal Snowflake review. If you are a Snowflake
employee, open a PR against the main branch and request review from the owner.

## Questions or issues

For questions, bugs, or feature requests, contact:

- **Owner**: john.kang@snowflake.com (GitHub: @sfc-gh-jkang)
